# AI聚合for超聚变

这是一个 Chrome 插件，允许你在一个页面中同时使用多个 AI 聊天机器人。

## 支持的 AI
- 豆包 (Doubao)
- DeepSeek
- 腾讯元宝 (Yuanbao)
- 通义千问 (Qianwen)

## 如何安装
1. 打开 Chrome 浏览器，进入 `chrome://extensions/`。
2. 开启右上角的“开发者模式”。
3. 点击“加载已解压的扩展程序”，选择此项目文件夹。

## 如何使用
1. 点击插件图标，选择“打开聚合面板”。
2. 在底部的统一输入框中输入消息。
3. 按 `Enter` 或点击“发送”，消息将同步发送到所有 AI 窗口。

## 注意事项
- **登录**：如果某个 AI 网站显示未登录，请直接在 iframe 中完成登录，或者在普通标签页登录后再刷新聚合面板。
- **验证码**：部分网站（如 DeepSeek）可能会触发验证码，需要在 iframe 中手动完成验证。
- **布局**：默认采用 2x2 网格布局。
